1 git push时，提示everthing up-to-date信息 *

上传没有成功

运行git push origin master时，提示everthing up-to-date,提交未成功。

解决办法： 执行git remote -v后看到自己的remote端名字为origin 

git remote -v

执行git branch后看到自己当下用的分支是master: 

git branch

因此在本地commit后，再push到remote端。 

git add -A
git commit -m "edit"
git push origin master

好了。没问题了。
